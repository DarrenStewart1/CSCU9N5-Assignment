Installing plugins

in eclipse you can install plugins to customize the way that your workspace or chance the functionality of your workspace in this tutorial I will show you how to install a dark theme from the eclipse market place this tutorial can be used to install other things from the market place aswell.

Written Guide

Step 1

The first thing you will need to do is open the market place to do go to the top of the screen and then press help it will bring up a menu then press eclipse market space this menu will then pop up.

Step 2

Next you will want to look for the plugin that you want to install for this tutorial I will be installing a dark theme use the search bar to find the plugin you want after finding the plugin you want to install press the install button the installation will start then you will be brough to a menu that what features you want to install then press confirm.

Step 3

You will then be asked to a licenses agreement to be able to use the plugin press I accept the terms of the license agreement then press finish

Step 4

You then need to restart eclipse for the plug in to get installed

Eclipse should then reset and the plugin will be installed sucefully
