Projects in eclipse

A project contains all the packages and classes that are to build the program.


Creating the project

Step 1

The first thing you will need to do is right click on the project manager (see the eclipse ui page if you don't know where that is you will then need to hover over new then click java project then this page will come up

Step 2

on the create java project page you will need to name the project aswell as change any setting is personaly recommend you use the default options once done press finish and this will create the project your project will appear in the project explorer

Step 3

Next you will need to create a new package this will hold the classes you make currently you will only use one package but as projects become more complex you will have more than one to create a new package to do this click on the arrow next to your project and then right click on src then hover over new then click create package then this window will appear

Step 4

similarly to when you created the project you will need to name the package it is recommended to start the name using a lower case letter once you have named the package click finish to create the package

Step 5

Next you will need to create a class to do this right click on the package you create then hover over new then press class to bring this menu up. next you will need to name class class you are creating it is recommended to start the name with an uppercase letter you will then want to select public static void main(String[] args)

Step 6

This is how the the menu should look before you press finish. After pressing finish you will now be able to start coding
